# Employee Management System

## Overview

Employee Management System is a desktop application developed using **C# WinForms** and **MySQL**. The system allows users to manage employee records, departments, salaries, and related information through a user-friendly interface.

## Features

* Add new employees
* Update employee information
* Delete employee records
* Search employees
* Department management
* Salary management
* MySQL database integration
* User-friendly Windows Forms interface

## Technologies Used

* C#
* .NET WinForms
* MySQL
* Visual Studio 2022

## Prerequisites

Before running the project, make sure you have:

* Visual Studio 2022 or later
* .NET SDK
* MySQL Server
* MySQL Connector for .NET

## How to Run

### 1. Clone the Repository

```bash
git clone https://github.com/dilshani-samudrika/Employee-Management-System.git
```

### 2. Open the Project

Open Visual Studio and select:

```text
Open a project or solution
```

Then open:

```text
Employee_managment_system.csproj
```

or

```text
Employee_managment_system.sln
```

### 3. Configure Database

* Create the MySQL database.
* Import the required SQL script.
* Update the connection string in the application.

### 4. Build the Project

In Visual Studio:

```text
Build → Build Solution
```

Shortcut:

```text
Ctrl + Shift + B
```

### 5. Run the Application

In Visual Studio:

```text
Debug → Start Debugging
```

Shortcut:

```text
F5
```
